# Roadmap MVP

## Fase 0 — Bootstrap documental

Criar repositório, documentação e ADRs.

## Fase 1 — Stack staging vazia/limpa

Criar `/opt/stacks/wp-loja-despertando` com WordPress, MariaDB, Redis e WP-CLI, sem exposição pública sem checkpoint humano.

## Fase 2 — WordPress + WooCommerce

Instalar WordPress, instalar WooCommerce e validar health básico.

## Fase 3 — Plugin base

Criar plugin `despertando-commerce-core`, registrar fulfillment types e adicionar estrutura modular.

## Fase 4 — MVP 1 Dimona

Integrar Dimona API com idempotência, logs, retry e reprocessamento.

## Fase 5 — MVP 2

Marketplace, Mercado Pago Split, afiliados e dropshipping.
