# Estratégia WooCommerce

WooCommerce foi escolhido por equilibrar velocidade de implantação, ecossistema brasileiro, pagamentos, frete, painel administrativo pronto e flexibilidade por plugin próprio.

## Regras

- Usar poucos plugins.
- Evitar plugins para regras centrais do negócio.
- Centralizar regra específica no `despertando-commerce-core`.
- Testar atualizações em staging.

## Fulfillment types

- `own_stock`.
- `dimona`.
- `marketplace`.
- `affiliate`.
- `dropshipping`.
