# Escopo do Produto

## Visão

A Loja Despertando será um e-commerce próprio, iniciado com produtos físicos próprios e print on demand via Dimona, evoluindo para marketplace, afiliados e dropshipping.

## MVP 1

Dentro do escopo:

- Catálogo de produtos.
- Produto físico próprio.
- Produto print on demand Dimona.
- Carrinho e checkout WooCommerce.
- Pagamento via Mercado Pago normal.
- Integração Dimona via API própria.
- Logs por pedido.
- Reprocessamento manual de integração Dimona.
- Proteção contra duplicidade de pedido Dimona.
- Staging na VPS.

Fora do escopo:

- Marketplace.
- Split de pagamento.
- Cadastro de vendedores externos.
- Produtos afiliados.
- Dropshipping.

## MVP 2

- Marketplace com vendedores externos.
- Mercado Pago Split.
- Produtos afiliados.
- Tracking de cliques.
- Dropshipping por fornecedor.
- Regras de carrinho por vendedor e tipo de produto.
