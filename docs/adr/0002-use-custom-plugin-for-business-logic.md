# ADR 0002 — Usar plugin próprio para regra de negócio

## Status

Aceita.

## Decisão

Criar o plugin próprio `despertando-commerce-core`.

## Motivos

- Evitar excesso de plugins para regra central.
- Centralizar domínio do negócio.
- Facilitar testes e manutenção.
- Permitir evolução modular por MVP.
