# ADR 0001 — Usar WooCommerce como motor de e-commerce

## Status

Aceita.

## Contexto

O projeto precisa vender produtos físicos próprios, print on demand com Dimona e futuramente marketplace, afiliados e dropshipping.

## Decisão

Usar WordPress + WooCommerce como motor principal de e-commerce.

## Motivos

- Menor tempo para vender.
- Ecossistema forte no Brasil.
- Checkout, carrinho, pedidos e painel prontos.
- Integrações de pagamento e frete mais acessíveis.
- Customização possível por plugin próprio.

## Consequências

O projeto herda cuidados do WordPress/WooCommerce. Atualizações devem ser testadas em staging.
