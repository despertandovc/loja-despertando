# Dropshipping — MVP 2

Permitir venda de produtos enviados por fornecedores terceiros.

## Fulfillment

```text
fulfillment_type = dropshipping
```

## Escopo inicial

- Cadastro de fornecedor.
- Produto associado ao fornecedor.
- Custo de compra.
- Margem.
- Prazo estimado.
- Envio de pedido para fornecedor.
- Registro de rastreio.
